# SaaS Customer Churn Prediction

Data-driven prediction engine leveraging subscription, usage, and support behavior data to model customer churn risk in SaaS businesses.

## Overview
This project builds an end-to-end machine learning pipeline to predict customer churn by analyzing behavioral patterns across subscriptions, product usage, and support interactions. The model identifies high-risk customers early, enabling proactive retention strategies.

## Key Features
- Data aggregation from multiple relational tables (accounts, subscriptions, usage, support)  
- Feature engineering from behavioral and billing signals  
- One-hot encoding for categorical variables  
- Random Forest classifier with hyperparameter tuning  
- Threshold optimization to balance churn recall and precision  

## Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Google Colab  
- KaggleHub
